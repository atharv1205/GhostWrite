> Your words. Your AI. Your machine.

A blazing-fast offline AI writing assistant built with **Rust**, **Tauri**, **React**, **Ollama**, and **Qwen 2.5**.

No subscriptions.

No API keys.

No cloud.

No one reading your embarrassing Slack messages.

Just local AI doing its thing.

---

## Why?

I got tired of constantly rewriting:

- Slack messages
- Emails
- LinkedIn posts
- Internship updates
- Documentation

…and I didn't want every sentence I typed traveling halfway across the internet.

So I built GhostWrite.

## ✨ Features

- ⚡ Global keyboard shortcut
- 🤖 Local AI powered by Qwen 2.5
- 🦀 Rust backend
- 🖥️ Native desktop app using Tauri
- 📋 Clipboard integration
- ✍️ Rewrite text
- 💼 Professional tone
- 😊 Friendly tone
- 📄 Summarize
- ✅ Fix grammar
- 🔒 Completely offline
- 🚀 No API costs

## Installation

Clone the repository

```bash
git clone https://github.com/atharv1205/GhostWrite.git
```

Install dependencies
```bash
npm install
```

Run
```bash
npm run tauri dev
```


---

## Requirements

- macOS
- Rust
- Node.js
- Ollama

Download the model

```bash
ollama pull qwen2.5:14b-instruct
```

## Privacy

GhostWrite promises to never:

- upload your text
- judge your grammar
- leak your secret situationship messages
- send your "hey bro" texts to OpenAI

Everything stays on **your machine**.
