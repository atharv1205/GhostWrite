# 🤝 Contributing to GhostWrite

First of all, thank you for considering contributing to **GhostWrite**! ❤️

Whether you're fixing a bug, improving the UI, optimizing performance, or adding new AI features—your contribution is appreciated.

---

## 🚀 Getting Started

### 1. Fork the repository

Click the **Fork** button on GitHub.

### 2. Clone your fork

```bash
git clone https://github.com/YOUR_USERNAME/GhostWrite.git
cd GhostWrite
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start Ollama

Make sure you have Ollama installed and the required model downloaded.

```bash
ollama pull qwen2.5:14b-instruct
```

### 5. Run the application

```bash
npm run tauri dev
```

---

# 🛠 Development Guidelines

Please try to follow these guidelines:

- Write clean and readable code.
- Keep commits focused on one change.
- Test your changes before opening a Pull Request.
- Follow the existing project structure.
- Prefer small, incremental improvements over massive rewrites.

---

# 💡 Good First Issues

Looking for something to work on?

Here are a few ideas:

- Improve the UI/UX
- Add dark mode enhancements
- Add support for more Ollama models
- Improve keyboard shortcuts
- Add settings/preferences
- Improve performance
- Add Linux support
- Add Windows support
- Improve documentation

---

# 📝 Commit Messages

Examples of good commit messages:

```text
feat: add grammar correction mode

fix: prevent clipboard overwrite on empty response

refactor: simplify ollama request handling

docs: improve installation guide
```

---

# 🔀 Pull Requests

When opening a Pull Request:

- Clearly describe what changed.
- Include screenshots if the UI changed.
- Link any related issues.
- Keep PRs focused on a single feature or fix.

---

# 🐛 Reporting Bugs

Found a bug?

Please include:

- Operating System
- GhostWrite version
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots (if applicable)

---

# ✨ Feature Requests

Have an idea?

Open an issue and explain:

- What problem you're trying to solve
- Your proposed solution
- Why it would improve GhostWrite

Great ideas often become the next release.

---

# ❤️ Code of Conduct

Be respectful.

Be constructive.

Be kind.

Everyone starts somewhere.
